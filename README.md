## 🌍 The Problem

Climate change is one of the greatest challenges of our generation — yet most people have **zero visibility** into how their daily habits impact the environment.

Ecoventra bridges that gap.

> **Make sustainability understandable, actionable, and rewarding.**

## 🛠️ Tech Stack

| Layer | Technologies |
|-------|-------------|
| **Frontend** | React 19, TypeScript, Tailwind CSS, Vite, Framer Motion, React Router, Recharts |
| **Backend** | Firebase Authentication, Firestore Database, Firebase Storage |

## 🎨 Design Philosophy

Ecoventra follows a modern **climate-tech design language** inspired by Apple, Stripe, Linear, Notion, and Tesla.

**Principles:** Minimal · Accessible · Data-Driven · Human-Centered · Sustainability-Focused
